Mean, Variance and Standard Deviation

Mean, Variance, and Standard Deviation are fundamental statistical measures used to describe the central tendency and dispersion of a random variable.

These measures are applicable to both discrete and continuous random variables and play a key role in probability distributions.

1. Discrete Random Variable

Let X be a discrete random variable with probability mass function P(X = xᵢ).

Formulas

Mean (Expected Value):

μ = E(X) = Σ xᵢ P(xᵢ)

Variance:

σ² = Var(X) = Σ (xᵢ − μ)² P(xᵢ)

Standard Deviation:

σ = √σ²

Solved Example

Given:

  • P(X = 0) = 0.2
  • P(X = 1) = 0.5
  • P(X = 2) = 0.3

Mean:

μ = (0)(0.2) + (1)(0.5) + (2)(0.3) = 1.1

E(X²):

E(X²) = (0²)(0.2) + (1²)(0.5) + (2²)(0.3) = 1.7

Variance:

σ² = 1.7 − (1.1)² = 0.49

Standard Deviation:

σ = √0.49 = 0.7

2. Continuous Random Variable

Let X be a continuous random variable with probability density function f(x).

Formulas

Mean:

μ = ∫ x f(x) dx

Variance:

σ² = ∫ (x − μ)² f(x) dx

Standard Deviation:

σ = √σ²

Solved Example (Uniform Distribution)

Let X be uniformly distributed over (0, 10).

f(x) = 1/10 , 0 ≤ x ≤ 10

Mean:

μ = (0 + 10)/2 = 5

Variance:

σ² = (b − a)² / 12 = (10 − 0)² / 12 = 8.33

Standard Deviation:

σ = √8.33 ≈ 2.88

3. Binomial Distribution

A binomial distribution models the number of successes in n independent trials, each with probability p.

Formulas

Mean: μ = np

Variance: σ² = npq

Standard Deviation: σ = √(npq)

Solved Example

n = 10, p = 0.4

μ = 10 × 0.4 = 4

σ² = 10 × 0.4 × 0.6 = 2.4

σ = √2.4 ≈ 1.55

4. Poisson Distribution

A Poisson distribution models the number of events occurring in a fixed interval when events occur randomly.

Formulas

Mean: μ = λ

Variance: σ² = λ

Standard Deviation: σ = √λ

Solved Example

λ = 3

μ = 3 , σ² = 3 , σ = √3 ≈ 1.73

Poisson Distribution - Example

Step 1: Estimate the Poisson Parameter (λ)

For a Poisson distribution, the mean is given by:

λ = Σfx / N

Compute Σfx:

Σfx = (0×32) + (1×15) + (2×9) + (3×4)

Σfx = 0 + 15 + 18 + 12 = 45

Total observations, N = 60

λ = 45 / 60 = 0.75


Step 2: Poisson Probability Formula

The probability mass function of Poisson distribution is:

P(X = x) = (e−λ λx) / x!

Substituting λ = 0.75:

P(X = x) = (e−0.75 (0.75)x) / x!

5. Normal Distribution

A normal distribution is a symmetric, bell-shaped distribution defined by mean μ and standard deviation σ.

Formulas

Mean = μ

Variance = σ²

Standard Deviation = σ

Solved Example

If X ~ N(50, 9):

μ = 50

σ² = 9

σ = √9 = 3

Solved Example: Normal Distribution

In a normal distribution, 31% of the items are under 45 and 10% of the items are over 64. Find the mean (μ) and standard deviation (σ).

Step 1: Given Data

  • P(X < 45) = 0.31
  • P(X > 64) = 0.10
  • X ~ N(μ, σ²)

Step 2: Convert to Standard Normal Variable

Standard normal variable:

Z = (X − μ) / σ

Step 3: First Condition

P(X < 45) = 0.31

⇒ P( (X − μ) / σ < (45 − μ) / σ ) = 0.31

From standard normal table:

Φ(z) = 0.31 ⇒ z = −0.50

Therefore,

(45 − μ) / σ = −0.50    (1)

Step 4: Second Condition

P(X > 64) = 0.10

⇒ P(X < 64) = 1 − 0.10 = 0.90

⇒ P( (X − μ) / σ < (64 − μ) / σ ) = 0.90

From standard normal table:

Φ(z) = 0.90 ⇒ z = 1.28

Therefore,

(64 − μ) / σ = 1.28    (2)

Step 5: Solve the Equations

From equation (1):

45 − μ = −0.50σ

μ − 0.50σ = 45    (3)

From equation (2):

64 − μ = 1.28σ

μ + 1.28σ = 64    (4)

Step 6: Final Calculation

Subtract equation (3) from equation (4):

(μ + 1.28σ) − (μ − 0.50σ) = 64 − 45

1.78σ = 19

σ = 10.67

Substitute σ into equation (3):

μ = 45 + 0.50(10.67)

μ = 50.34

Final Answer

  • Mean (μ) = 50.34
  • Standard Deviation (σ) = 10.67
