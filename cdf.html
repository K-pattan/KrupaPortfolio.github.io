It is denoted by F(x) and is defined for both discrete and continuous random variables.

Solved Example (Discrete CDF)

Problem:

A random variable X takes the following values:

  • P(X = 1) = 0.1
  • P(X = 2) = 0.2
  • P(X = 3) = 0.3
  • P(X = 4) = 0.25
  • P(X = 5) = 0.15

Find: The cumulative distribution function F(x).

Solution:

  • F(1) = 0.1
  • F(2) = 0.1 + 0.2 = 0.3
  • F(3) = 0.6
  • F(4) = 0.85
  • F(5) = 1.0
