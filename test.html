Hypothesis Testing Using Real Data Sets

Hypothesis testing is a statistical method used to make decisions about a population parameter based on sample information. In real-world applications, data collected from industries, experiments, or surveys is used to verify claims.

In this example, we apply a Z-test for mean using real production data from a manufacturing company.

Problem Statement (LP-16)

Python Code Used for Computation

import numpy as np
from scipy.stats import norm

s_mean = 1570
p_mean = 1600
std = 120
s_size = 100

z = (s_mean - p_mean) / (std / np.sqrt(s_size))
