Probability Mass Function (PMF)

A Probability Mass Function (PMF) describes the probability distribution of a discrete random variable. It assigns a probability to each possible value the random variable can take.

PMFs are widely used in probability theory, statistics, machine learning, queueing theory, and reliability engineering.

Mathematical Definition

Let X be a discrete random variable. The Probability Mass Function is defined as:

P(X = x) = f(x)

where:

  • f(x) ‚â• 0 for all x
  • Œ£ f(x) = 1 (sum over all possible x)

Key Properties of PMF

  • Defined only for discrete random variables
  • Each probability lies between 0 and 1
  • Total probability equals 1
  • Graph is represented using separate vertical bars

Worked Example (Solved)

Problem:

A fair die is thrown once. Let X be the number obtained.

Solution:

Possible values of X are:

X = {1, 2, 3, 4, 5, 6}

Since the die is fair:

P(X = x) = 1/6 for each x

This defines the PMF of X.

PMF Table for the Die Example

P(X = x) 1/6 1/6 1/6 1/6 1/6 1/6

Graphical Representation of PMF

The graph of a PMF consists of vertical bars at discrete values of X. The bars do not touch each other, emphasizing the discrete nature.
