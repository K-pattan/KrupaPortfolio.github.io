Random Variables

A random variable is a numerical quantity whose value depends on the outcome of a random experiment. Random variables are used to quantify uncertainty in probabilistic and statistical analysis.

Random variables are broadly classified into:

  • Discrete Random Variables
  • Continuous Random Variables

Discrete Random Variables

A discrete random variable takes a finite or countably infinite number of distinct values. These values can be listed individually.

Examples:

  • Number of heads obtained when tossing coins
  • Number of defective items in a batch
  • Number of students present in a classroom

Continuous Random Variables

A continuous random variable can take any value within a given interval. The number of possible values is infinite and cannot be listed individually.

Examples:

  • Height of students
  • Time taken to complete a task
  • Temperature of a city
