Histogram and Box Plot using Python

Graphical representation of data is essential in Probability and Statistics to understand distribution, spread, and variability of data. Two important graphical tools are the Histogram and the Box Plot.

1. Histogram

A Histogram is a graphical representation of the frequency distribution of continuous data. It divides data into class intervals (bins) and shows how many observations fall into each interval.

Example Dataset

45, 50, 52, 55, 58, 60, 62, 65, 68, 70

Python Code

import matplotlib.pyplot as plt


marks = [45, 50, 52, 55, 58, 60, 62, 65, 68, 70]


plt.hist(marks, bins=5, edgecolor='black')
plt.xlabel("Marks")
plt.ylabel("Frequency")
plt.title("Histogram of Student Marks")
plt.grid(axis='y', linestyle='--', alpha=0.7)
plt.show()

Histogram Graph

Interpretation:
The histogram shows the distribution of marks across intervals. It helps identify the shape of the data (symmetry or skewness) and concentration of values.

2. Box Plot

A Box Plot (Box-and-Whisker plot) represents data using five-number summary: minimum, first quartile (Q1), median, third quartile (Q3), and maximum.

Python Code

import matplotlib.pyplot as plt
marks = [45, 50, 52, 55, 58, 60, 62, 65, 68, 70]

plt.boxplot(
    marks,
    vert=True,
    patch_artist=True
)

plt.ylabel("Marks")
plt.title("Box Plot of Student Marks")
plt.grid(axis='y', linestyle='--', alpha=0.6)
plt.show()

Box Plot Graph

Interpretation:
The line inside the box represents the median. The box shows the interquartile range (IQR), and whiskers indicate the spread of data. Any points outside whiskers represent outliers.

Histogram vs Box Plot

Histogram Box Plot
Shows frequency distribution Shows quartile summary
Useful for shape analysis Useful for detecting outliers
Best for large datasets Best for comparison
