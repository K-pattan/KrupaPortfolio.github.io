Confidence Intervals

A confidence interval is a range of values used to estimate an unknown population parameter such as mean or proportion.

It provides an interval within which the true population parameter is expected to lie with a specified level of confidence.

Common confidence levels are 90%, 95%, and 99%.

Important Terms

  • Point Estimate: Sample statistic (mean or proportion)
  • Confidence Level: Probability that the interval contains the true parameter
  • Margin of Error: Maximum possible error in estimation
  • Critical Value: z or t value based on confidence level

General Formula

Confidence Interval = Point Estimate ¬± Margin of Error

Margin of Error = Critical Value √ó Standard Error
