Types of Data and Variables

In statistics, data refers to raw facts, figures, or observations collected for analysis. Data helps in understanding patterns, drawing conclusions, and making informed decisions in engineering and real-world applications.

Types of Data

Based on the nature of information, data is classified into the following types:

Qualitative (Categorical) Data

Represents non-numerical characteristics or categories.

  • Gender of students (Male, Female)
  • Department (CSE, ECE, ME)
  • Blood group (A, B, AB, O)

Quantitative (Numerical) Data

Represents numerical values that can be measured or counted.

  • Marks obtained in an exam
  • Height of students (in cm)
  • Time taken to complete a task (in seconds)

Types of Variables

A variable is a characteristic that can take different values for different observations. Variables are classified as follows:

Discrete Variables

Take countable values.

  • Number of students in a class
  • Number of defective items in a batch
  • Number of calls received in a day

Continuous Variables

Can take any value within a given range.

  • Temperature of a room (°C)
  • Weight of a person (kg)
  • Time taken to run a race (seconds)

Importance of Classification

Understanding the types of data and variables is essential for selecting appropriate statistical techniques, probability distributions, and graphical representations. This classification forms the foundation for advanced topics in probability, statistics, data science, and machine learning.
