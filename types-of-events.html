Types of Events in Probability

An event in probability is a set of outcomes of a random experiment. Understanding different types of events helps in selecting the correct probability rules and modeling real-world uncertain situations.

1. Equally Likely Events

Events are said to be equally likely when each event has the same probability of occurrence.

Example:

  • Experiment: Rolling a fair die
  • Sample space: {1, 2, 3, 4, 5, 6}
  • Each outcome has probability = 1/6

Equally likely events are commonly used in classical probability problems.

2. Complementary Event

Example:

  • Event A: Getting a head in a coin toss
  • P(A) = 1/2
  • P(A‚Ä≤) = 1 ‚àí 1/2 = 1/2

Complementary events help simplify probability calculations.

3. Mutually Exclusive (Disjoint) Events

Two events are said to be mutually exclusive if they cannot occur at the same time.

Property:

P(A ‚à© B) = 0

Example:

  • Experiment: Tossing a coin
  • Events: Head (H) and Tail (T)
  • H and T cannot occur together

Mutually exclusive events are important in applying the addition theorem of probability.

4. Independent Events

Two events are independent if the occurrence of one event does not affect the probability of the other event.

Formula:

P(A ‚à© B) = P(A) √ó P(B)

Independent events are widely used in probability models and machine learning.

5. Dependent Events

Two events are dependent if the occurrence of one event affects the probability of the other event.

Formula:

P(A ‚à© B) = P(A) √ó P(B | A)

Example:

  • Experiment: Drawing two cards successively without replacement
  • Event A: First card is an Ace
  • Event B: Second card is an Ace
  • P(B | A) = 3/51

Dependent events are commonly encountered in sampling without replacement and real-world decision processes.

Summary

Understanding these types of events helps in applying probability laws correctly and modeling uncertainty effectively in engineering and data science.
